# checkboxes-in-list-view

## Checkbox and checkboxToggle
Render the label of the item instead of "Yes" when checked

## CheckboxLabeledToggle
Render the labelChecked and labelUnchecked instead of Yes/ No
https://docs.typo3.org/m/typo3/reference-tca/main/en-us/ColumnsConfig/Type/Check/LabeledToggle.html

## Multiple checkboxes of any kind
Use TYPO3 default behaviour: render comma-list of checked labels
